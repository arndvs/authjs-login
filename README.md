```bash
npm run dev

```

Generate the Prisma collections based on schema.prisma

```bash
npx prisma generate

```

Push the collections to the db

```bash
npx prisma db push

```

Run Prisma Studio to view the current prisma database

```bash
npx prisma studio

```

More Prisma Commands: https://www.prisma.io/docs/orm/reference/prisma-cli-reference#studio
